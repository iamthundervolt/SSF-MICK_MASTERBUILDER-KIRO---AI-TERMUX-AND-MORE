# HTML MASTERBUILDER

A high-end, interactive, lightweight HTML-based system optimized for iPad/iPhone with 7 powerful tabs.

## Features

### 1. Main - System Overview
Live metrics dashboard showing CPU, memory, tasks, and network status

### 2. Live Terminal - UNIFIED AI AGENT
Revolutionary AI chat agent with "The Soul" - Unified AI Bridge
- **World's Greatest Termux Expert** with deep knowledge base
- 20+ Termux packages (Python, Node.js, Git, SSH, Nginx, databases, etc.)
- 10+ Termux APIs (battery, camera, location, notifications, etc.)
- Hybrid Intelligence: Works brilliantly with OR without API keys
- Multi-source AI: Automatically tries Groq, HuggingFace, OpenAI
- Context-aware responses enriched with Termux expertise
- Commands: help, chat, termux, about, config, system

### 3. Blockchain Scanner
Real-time blockchain scanning with start/stop controls
- Live block monitoring
- Transaction tracking

### 4. Telemetry
Full sensor suite with GPS mapping
- Temperature, altitude, air pressure
- Google Maps integration (requires API key)

### 5. Pterodactyl Game
Flappy Bird style game with pterodactyl
- Touch/click to jump
- Score tracking

### 6. Music Studio
Complete music production suite
- Drum machine with 16-step sequencer
- Synthesizer with step sequencer
- Bassline groove box
- Record and export functionality

### 7. Teletext News
Live news feed using teletext format
- Page navigation
- Interactive display

## Installation

1. Open `index.html` in any modern browser
2. For iOS: Open in Safari, tap Share, then "Add to Home Screen"
3. The app will launch fullscreen from your home screen

## Customization

### Google Maps API
Replace `YOUR_API_KEY` in `app.js` line 186 with your Google Maps API key

### Styling
Edit `styles.css` to customize colors, gradients, and layout

### Adding Features
Use `HTML_MASTER_BUILDER_TABSTYLE.html` as a template for new projects
